How it works

  • • Derives a key with PBKDF2 and a random salt.
  • • Encrypts with AES-GCM using a secure IV.
  • • Outputs file as downloadable blob with embedded salt and IV.
  • • Decryption reads salt/IV and verifies authenticity automatically.

Encryption file format

The encrypted output embeds the salt and IV in the first bytes, so decryption can reconstruct the same key and verify the ciphertext.
